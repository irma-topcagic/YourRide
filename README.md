# 🚗 YourRide – ASP.NET Web Application

### 📌 Project Overview
**YourRide** is a robust web application built using the **ASP.NET Core** framework. The project focuses on clean architecture, object-oriented design, and thorough system modeling. It was developed as a collaborative effort to demonstrate advanced software engineering principles.

---

### 🏗 Architecture & Design
This project is heavily focused on design quality. It includes comprehensive documentation of the system's architecture, including:

* **Creational, Structural, and Behavioral Patterns** (SOLID principles, Decorator, Composite, etc.).
* **UML Modeling**: Use Case, Class, Sequence, and Activity diagrams.
* **Database Design**: Detailed Entity Relationship Diagram (ERD).

---

### ✨ Features
* **User Management**: Identity-based authentication and authorization.
* **Ride Management**: Core logic for handling transport requests and processing.
* **MVC Pattern**: Clear separation of concerns between Models, Views, and Controllers.

---

### 🛠 Technologies
* **C# / .NET Core**
* **Entity Framework Core** (SQL Database)
* **ASP.NET MVC & Razor Pages**
* **HTML5 / CSS3 / JavaScript**

---

### 📁 Documentation
All project diagrams and architectural specifications can be found in the [**Dokumentacija**](./Dokumentacija) folder. These files cover the complete software development lifecycle (SDLC) from analysis to design patterns implementation.

---

### 🚀 How to Run
1.  Clone the repository.
2.  Open the solution file (`.sln`) in **Visual Studio**.
3.  Ensure you have the **.NET SDK** installed.
4.  Run `Update-Database` in the Package Manager Console.
5.  Press `F5` to run the application.
